--Setup:
Για να τρέξειι η εφαρμογή πρέπει:
1) Να υπάρχει κάποια έκδοση της Python εγκατεστήμενη στον υπολογιστή
2) Να γίνουν pip install οι παρακάτω βιβλιοθήκες της Python:

pip install pillow
pip install navigator
pip install qrcode
pip install datetime
pip install Image
pip install messagebox

-->Eδώ μπορείτε, σε περίπτωση που το επιθυμείτε, και εφόσον η Python έχει προστεθεί
στο Path, στα User και System Enviroment Variables, να τρέξετε το .bat αρχείο 
το οποίο αναλαμβάνει να ενημερώσει το pip installer του υπολογιστή σας και να
εγκαταστήσει τις παραπάνω βιβλιοθήκες, χωρίς να χρειαστεί να το κάνετε χειροκίνητα
-->Κατά πάσα πιθανότητα τα Windows, θα σας αποτρέψουν λόγο αγνώστου προέλευσης αρχείου,
μπορείτε να το παρακάμψετε με δική σας ευθύνη

3) Για να τρέξει σώστα το πρόγραμμα πρέπει όλα τα αρχεία .py, 
το αρχείο upEAT_1.sqlite καθώς και οι φάκελοι digital_cards,media,profpic,qrcodes να βρίσκονται
σε έναν ενιάιο φάκελο.

-->Ιδιοτροπία του VSCode: ο τρόπος που αντιμετώπιζει τα relative paths το VSCode, είναι
ιδιαίτερως και έτσι θέλει ΠΡΟΣΟΧΗ, σε περίπτωση που τρέξετε μέσω αυτού. Πρέπει να ανοίξετε
"OPEN FOLDER" τον φάκελο που περίεχει τα προαναφερθέντα και όχι κάποιον φάκελο σε υψηλότερο επίπεδο

4) Εκτελέστε το main.py και το πρόγραμμα θα ανοίξει

Μπορείτε να δημιουργήσετε δικό σας λογαριασμό μέσω του Register ή να αξιοποιήσετε τον ειδικά δημιουργήμενο λογαριασμό:

username: demo
password: demo 

που θα σας επιτρέψει να δείτε όλες τις λειτουργίες του προγράμματος. Σε αντίθεση με τον λογαριασμό που θα δημιουργήσετε εσείς και δεν
θα είναι γεμάτος με δεδομένα.
